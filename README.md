Repo for terminal config

# Installation
1. Install a nerd font and activate in terminal https://www.nerdfonts.com/

2. Install zsh + oh-my-zsh `sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"`

3. Clone this repo to ~/.zsd/ `git clone git@github.com:fnyger/zsh_conf.git ~/.zsh/`

4. Run install script `source ./install.sh`
